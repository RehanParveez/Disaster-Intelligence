## Disaster Intelligence:

This backend system project is about handling the real time disaster situations like floods, earthquakes, and other big emergencies. In this project the effort to build the functionality of choosing which incident needs the attention first is present. Further in it the process of allocating resources intelligently with the use of a scheduler-driven decision engine is also present. So in this project the core attempt is to design a system where the whole workflow is not majorly predefined rather its an attempt to make decisions in real-time which is based on changing conditions.
